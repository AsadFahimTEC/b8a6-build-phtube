Discuss the scope of var, let, and const

The scope of var is a functional scope. It means that variable defined the outside function can be accessed globally
and variable defined inside a particular function can be accessed within a function. Let's here given an example:
var a = 29; // here also output is 29
function sum( ){
console.log(a); // output is 29
} sum( ); //
we can also redeclare var value but not declare before initialization.


The scope of let and const is a block scope. For let we cannot access its value outside the particular block. Let's here given example:
let a = 29;
function sum( ){
let b = 10;
console.log('the value of b is', b) // b prints value 10
console.log('the value of a is', a); // a prints value 29
} sum( ); let a = 29;

let b = 10;

}

} sum( );

// the value of a is 29
